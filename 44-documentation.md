# Documentation

- Public comments **must** be included over every component.
- Services **should** have descriptive comments.
- Comments **should** be written in correct English.
  - The first sentence **should** omit the subject and be in third-person present tense.
- Descriptions **should** be brief but complete.
- Comments **may** link to external pages to provide background.
- External links **must** use absolute URLs.
- To deprecate a component, deprecated **must** be set to true. The comment **must** start with `Deprecated:`.
